# Student Feedback System
This is a web-based Student Feedback System developed using Java and MySQL.
It allows students to submit feedback and helps teachers/admin to view and manage feedback easily.
## Features
- Student can submit feedback
- Admin can view feedback
- Simple and user-friendly interface
- Secure data storage using MySQL
## Technologies Used
- Java
- sql
- servlet
- MySQL
  
## How to Run
1. Install eclipse
2. Copy the project folder to htdocs
3. Start Apache and MySQL
4. Import database file into phpMyAdmin
5. Open browser and run: http://localhost/student-feedback-system
## Future Scope
- Login system for students and teachers
- Feedback analytics and charts
- Email notification system
